Welcome!

This page is for collecting ideas and suggestions from students of KIMSS S1 Class of 2024 to help us design and build the official class website. Everything you write here is meant to improve how our class is seen, remembered and enjoyed online.

✔ These suggestions are only for the class website ✔ Everyone in class can give ideas ✔ The final website will be based on what students suggest here

About This Suggestion Project

The goal of this project is to build a class website that truly represents ourclass- our vibes, jokes, photos, memories, achievements and stories. Instead of one person deciding everything, this page lets the whole class contribute.

By participating, you help decide:

  • The best class name to use on the website
  • Which sections and categories should be on the site
  • What kind of content (jazz, photos, events, memories, etc.) will appear
  • How KIMSS S1 Class of 2024 will be remembered in future

Every clean and respectful idea is welcome – serious, funny or creative. Your voice matters. Help us create a class website we’ll be proud to show off.
